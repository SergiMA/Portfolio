About me

My name is Sergi Monzó. I'm a game developer specialised in gameplay and AI programming. But I also have experience with sound and UI programming.

I love coding and I'm always trying to learn new things. I've always loved videogames and one of my wishes is to make them. I started out studying programming using C and Java as my main programming languages for my studies. And while developing my very first videogame project at class, I decided to study game development professionally. Then I moved to C# and Unity. And here I am: making videogames with Unity engine.

I have plans to move to other game engines, like Unreal Engine, because I want to know how it feels to make videogames in other engines and, depending on the projects I want to do, I want to be able to decide which engine could be better depending on the needs of the project.
